# HiT_web


Pour commencer : 

Créer la db:

    python3 manage.py migrate
    
    python3 manage.py createsuperuser

Lancer le serveur:

    python3 manage.py runserver

Puis dans localhost:8000/admin:

    créer un challenge (ex : docker_name = ex_1)

et hop :D
